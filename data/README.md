# Data

This folder is empty by default. Download the datasets 
and place them here following this structure:

data/vidvrd/
  annotations/
    train/
    test/
    test_gt.json
  videos/

data/vidor/
  annotation/
    training/
    validation/
  videos/

data/action_genome/
  annotations/
  videos/

Download links:
- VidVRD: https://xdshang.github.io/docs/imagenet-vidvrd.html
- VidOR: https://xdshang.github.io/docs/vidor.html  
- Action Genome: https://prior.allenai.org/projects/charades
